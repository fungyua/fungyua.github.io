  2. PHP 变量

    • 变量的命名规则

      变量必须以美元符号 $ 开头，美元符号 $ 后面的第一个字符不能为数字，只能是下划线_或者字母，除了下划线_外，变量不允许出现任何空格或者标点符号，php 变量名是严格区分大小写的。

    • 变量的数据类型

      标量类型：boolean、integer、float、string
      复合类型：array、object
      两种特殊类型： resource（资源），NULL (无类型)

  3. PHP 常量

    • 自定义常量

      PHP 定义常量使用 define () 方法来定义
